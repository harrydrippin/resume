BinaryVR Face Manager

BinaryVR Face Manager는 Deep Learning Data Management Web Toolset이며, 딥러닝 데이터에 대한 관리와 Mechanical Turk Worker들의 Data Annotation Tool을 포함합니다. 딥러닝 모델의 실패 케이스를 모아서 정리하고 다시 학습하는 등의 관리가 가능하며, 약 9개 이상의 Tool들을 개발하였습니다.

Asynchronous Processing 도구의 목적과 특성 상 로딩 시간이 길면 그만큼 작업 속도가 저하되는 구조이기 때문에 최적화가 매우 중요한 이슈였습니다. 데이터를 최대한 비동기적으로 처리하여 로딩 시간을 최소화하는 작업을 하는데 많은 공을 들였습니다.
SPA Web Client 개발 React.js를 이용하여 Toolset 전체를 개발했습니다. 서버와의 통신은 REST API와 WebSocket을 주로 사용했습니다.
Image Processing과 Web Server Backend Node 10을 사용하여 서버를 개발했으며, OpenCV의 Typescript 포팅 버전을 사용하여 수 많은 얼굴 사진들의 Boundary를 인식하여 사용자에게 보여줄 수 있도록 자르는 등의 정제 작업을 실시했습니다.
BinaryFace for Web

BinaryFace는 C++ 기반의 Realtime Face Tracking Tool입니다. 카메라를 통하여 영상을 받고, 학습된 모델을 통해 84개의 Landmark를 얼굴에 배치합니다. 이 툴을 Web 환경에서도 사용할 수 있도록 연구하여 개발했습니다.

WebRTC를 통한 영상 전송 및 Feedback WebRTC Protocol을 약간 수정하여 이용해서 서버에 있는 모델과 사용자들을 비동기 P2P Direct Connection으로 연결하고 영상을 전송하는 방식으로 구현했습니다. 서버의 위치나 네트워크 상황 등의 다양한 변수로 인하여 영상이 열화되거나 Delay가 발생하는 등의 문제가 있었습니다. 이 문제는 하술할 다음 구현에서 해결했습니다.
WebAssembly를 통한 On-Browser Model Emscripten으로 빌드된 WebAssembly 모델을 사용하여 React.js와 연동했습니다. WebRTC 함수들을 통해 영상을 확보하고 실시간으로 브라우저 위에서 영상을 처리하여 사용자에게 보여줄 수 있는 환경을 구축하였습니다. 열화 현상이나 영상 지연 없이 빠르게 Web 환경에서 접근할 수 있도록 구현하였습니다.
